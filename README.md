This repository was created in 2019 during a Summer Research Experience for Undergraduates (REU) in Physics.

The goal of the code base was to understand different particle physics kinematics by writing algorithms in C and C++ for fast processing of signal samples. The Monte Carlo signal processes were in .root format that required pre-processing to retrieve necessary information for the project.

The exercise performs the reconstruction of a new parent particle at different mass points and the efficiency of the reconstruction method.
